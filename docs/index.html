Documentación de la Prueba Técnica

Bienvenido a la documentación de la API desarrollada para la prueba técnica . Aquí encontrarás enlaces a la documentación técnica generada y al esquema OpenAPI.

  • JavaDocs - Documentación detallada de la API en Java.
  • OpenAPI - Especificación de la API en formato YAML (Ver más).
  • Resultados de las Pruebas - Resultados de pruebas realizadas en Postman (Ver más).

Las colecciones de Postman para pruebas de la API están disponibles en el repositorio. Puedes descargarlas desde la carpeta raíz del proyecto.

		servers:

		tags:
